# Generated Templates Folder

> [!WARNING]
> **DO NOT EDIT ANY FILES IN THIS DIRECTORY DIRECTLY.**
> 
> All files in `src/templates/` are **automatically generated** by the template compiler script. Any manual modifications here will be overwritten the next time the CLI is run or built.

## How to make changes:
1. Make your edits to the raw files inside the root **`templates/`** directory.
2. Run the compiler script to update this folder:
   ```bash
   bun run compile-templates
   ```
   *(Or simply run `bun run dev` which compiles templates automatically before running the CLI).*
